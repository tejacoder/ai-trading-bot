## 🎯 Setup 5 Agents

### Super Quick Script

```bash
#!/bin/bash
# setup_5_agents.sh

REPO="https://github.com/tejacoder/ai-trading-bot.git"

for i in {1..5}; do
  echo "🤖 Setting up Agent $i..."
  
  git clone $REPO agent$i
  cd agent$i
  python3 setup_auto.py
  cd ..
  
  echo "✅ Agent $i done!"
done

echo "🎉 All 5 agents ready!"
```

**Usage:**
```bash
chmod +x setup_5_agents.sh
./setup_5_agents.sh
# ✅ 5 agents in ~2 minutes!
```
